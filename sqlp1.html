Summary of COVID-19 SQL Project

This SQL project focuses on analyzing COVID-19 data from two main datasets: one containing information about COVID-19 deaths (`PortfolioProject..CovidDeaths$`) and another containing data on COVID-19 vaccinations (`PortfolioProject..CovidVaccinations$`). Here's a summary of the project's SQL queries and actions:

  1. The project begins by selecting and ordering data from the COVID-19 deaths and vaccinations datasets, filtering out records where the continent is not specified.
  2. Various queries are then executed to analyze different aspects of the COVID-19 data. These include:
    • Calculating death percentages for Ghana.
    • Analyzing total cases versus population to determine the percentage of population infected.
    • Identifying countries with the highest infection rates compared to their populations.
    • Identifying countries with the highest death counts per population.
    • Analyzing death counts per population for each continent.
    • Calculating global COVID-19 statistics such as total cases, total deaths, and death percentages over time.
    • Investigating the relationship between total population and vaccination data.
  3. Common techniques such as using common table expressions (CTEs), temporary tables (`#PercentPopulationVaccinated`), and creating a view (`PercentPopulationVaccinated`) are utilized for data manipulation and analysis.
  4. The final step involves creating a view (`PercentPopulationVaccinated`) to store the processed data for potential later use in visualizations or further analysis.

Overall, this SQL project aims to provide insights into the spread of COVID-19, its impact on different countries and continents, and the progress of vaccination efforts worldwide.
